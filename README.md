# SimpleOpenGLVoxelGameDemo
一个使用glfw和OpenGL开发的体素游戏演示，使用了c++23进行开发，编写了大量的容器模板以及3d图形所使用的数学库，经测试通常情况下性能要比std::map std::unordered_map更好。
目前已在windows10, Visual studio 2022 环境下测试并编译运行成功

## 操作说明
- 移动：`W/A/S/D`
- 跳跃：`空格键`
- 物品快捷栏：`1` ~ `0`（大键盘1~0）或 `鼠标滚轮`
- 破坏方块：`鼠标左键`
- 放置方块：`鼠标右键`
- 切换飞行模式：`F`
- 向上飞行：`空格键`
- 向下飞行：`左ctrl`
- 切换穿墙模式：`C`
- 切换显示鼠标光标：`E`
- 切换全屏：`F11`
- 退出游戏：`Esc`
